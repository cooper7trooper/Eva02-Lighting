# Basic Lighting Code For EVA Unit 2 Helmet

## Compatibility
This code is designed to run on a ESP32-C3 and control 32 ws2812b individually addressible LED's.

## Pin Layout
 - Pins 4 and 5 are for each side of the face. 4 is for the left eyes and 5 is for the right eyes.
 - Pin 0 is for a push button and allows you to cycle through differen modes.
